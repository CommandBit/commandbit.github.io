Components

This page provides a visual reference for Bootstrap components and Bootswatch theme settings as used by the Felmdrav theme.

It is intended to help you evaluate typography, colors, form elements, buttons, and other UI components across different Bootstrap and Bootswatch themes.

Note: For demonstration purposes, this page intentionally uses the fixed-top navbar header variant instead of the hero header.

This makes it easier to focus on component styling without the visual influence of a hero image.

All components shown here follow standard Bootstrap behavior.
The theme does not introduce custom component styling beyond layout and spacing.
